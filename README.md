## Tech Stack

**Client:** React JS

**Server:** Firebase

## Project initialization

Clone the project

```bash
    mkdir taskerr
    cd taskerr
    https://github.com/rishav2803/Taskerr.git
```
Install dependencies

For the Client
```bash
  cd taskerr
  npm install
```

Start the Client

```bash
  cd taskerr
  npm start
```
